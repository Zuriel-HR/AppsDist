# Práctica de Repaso - Aplicaciones Distribuidas

UPIITA-IPN

Este proyecto contiene 10 servicios web desarrollados en Node.js para practicar conceptos de servicios REST y manejo de datos en memoria.

## Ejercicios implementados

1. **Servicio de Saludo Básico**: POST /saludo
2. **Calculadora de Operaciones Básicas**: POST /calcular
3. **Gestor de Tareas (CRUD Básico)**: POST/GET/PUT/DELETE /tareas
4. **Validador de Contraseñas**: POST /validar-password
5. **Conversor de Temperatura**: POST /convertir-temperatura
6. **Buscador en Array**: POST /buscar
7. **Contador de Palabras**: POST /contar-palabras
8. **Generador de Perfiles de Usuario**: POST /generar-perfil
9. **Sistema de Calificaciones**: POST /calcular-promedio
10. **API de Productos con Filtros**: GET/POST /productos

## Uso

1. Instala las dependencias:

```powershell
npm install express body-parser uuid
```

2. Ejecuta el servidor:

```powershell
node index.js
```

3. Realiza peticiones a los endpoints según la descripción de cada ejercicio.

## Evidencias

Coloca capturas de pantalla de pruebas en la carpeta `docs/`.
